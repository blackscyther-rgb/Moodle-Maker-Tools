Basic instructions

Ensure you follow the pre-requisite setup guide before implementing this navigation!

  1. Start filling in the relevant fields - they will update the preview dynamically
  2. Move between the tab (sections) to change different settings
  3. Click 'copy code', to copy the generated combined HTML+JS code to memory, and then paste it in the appropriate area
  4. If you're stuck, be sure to read the guide and watch the video which gives an overview of use case, and intended practice
